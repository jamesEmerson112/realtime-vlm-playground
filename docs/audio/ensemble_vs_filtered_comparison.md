## Analysis

### Why the delta is small

The existing `run_ensemble()` already calls `is_hallucination()` to exclude hallucinated
transcripts from the valid pool before voting. The filtered ensemble converts them to
NO_SPEECH *votes* instead of simply excluding them, but since conservative models
(gemini-3-flash, gpt-4o-audio-preview) already vote NO_SPEECH on silence, the outcome
rarely changes.

### False positive risk

The pattern `"your class is over"` matches real instructor speech in R066 170-175s
(instructor ending the procedure). This converts a correct 2-model agreement into GARBAGE.
Recommend removing `"your class is over"` and `"the end"` from HALLUCINATION_PATTERNS —
these are plausible classroom/procedure phrases.

### Conclusion

Filtered ensemble provides negligible uplift over plain ensemble on this dataset.
The main value of the expanded pattern list is for the `passes` method (single-model
hallucination filtering). For the ensemble, the voting mechanism already handles
hallucinations effectively.
